## Prosjekt

Bridge er en digital samhandlingsplattform for EDI-kommunikasjon, transportstyring og datadrevet samarbeid mellom leverandører, transportører, grossister og detaljister.

Bridge Flow standardiserer EDI-utveksling og reduserer manuelt administrativt arbeid. Bridge Transport fungerer som et moderne Transport Management System for planlegging, utførelse og oppfølging av transporter, med sanntidssporing, avviksbehandling og analyse.

## Rolle og bidrag

- Bygget Bridge Transport og felles plattformfunksjonalitet fra grunnen av.
- Utformet og implementerte komponentarkitektur, visuelt system og sentrale transportflyter i React, TypeScript, SCSS, React Hook Form, TanStack Query og TanStack Table.
- Bidro til Kotlin/Spring Boot-tjenester for turer, rutehåndtering, transport, booking og analyse.
- Arbeidet med automatiserte bygg, tester og container-deployments til Azure Kubernetes Service via Azure Pipelines.
- Sikret kvalitet med Jest, JUnit, MockK, Mongock og Testcontainers.
- Implementerte logging og sporing i Datadog og LogRocket for bedre innsikt fra frontend til backend.

## Project

Bridge is a digital collaboration platform for EDI communication, transport management and data-driven collaboration between suppliers, carriers, wholesalers and retailers.

Bridge Flow standardizes EDI exchange and reduces manual administration. Bridge Transport works as a modern Transport Management System for planning, executing and following up deliveries, including real-time tracking, deviation handling and analytics.

## Role and contribution

- Built Bridge Transport and shared platform functionality from the ground up.
- Designed and implemented the component architecture, visual system and key transport workflows in React, TypeScript, SCSS, React Hook Form, TanStack Query and TanStack Table.
- Contributed to Kotlin/Spring Boot services for trips, route handling, transport, booking and analytics.
- Worked on automated builds, tests and container deployments to Azure Kubernetes Service through Azure Pipelines.
- Improved quality with Jest, JUnit, MockK, Mongock and Testcontainers.
- Implemented logging and tracing in Datadog and LogRocket to improve insight from frontend to backend.
:::
