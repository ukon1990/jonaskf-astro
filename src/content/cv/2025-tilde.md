## Prosjekt

Tilde er Miljødirektoratets nye plattform for å erstatte eldre fagsystemer innen forurensning og miljøforvaltning. Løsningen bygges som mikrotjenester på Azure, med tjenester i Docker-containere på Kubernetes og leveranser gjennom GitHub Actions og FluxCD.

Plattformen dekker flere brukerflater: eksterne brukere som sender inn søknader og rapporter, saksbehandlere som behandler saker og gjennomfører tilsyn, og administratorer som forvalter skjemaer, kodeverk og tilgang.

## Rolle og bidrag

- Hovedsakelig frontendutvikling i Angular, TypeScript og SCSS.
- Brukerflater for søknader, rapportering, tilsyn og saksbehandling på tvers av portalene.
- Komponenter og løsninger basert på Miljødirektoratets designsystem og delte Angular-biblioteker.
- Samarbeid med backendutviklere og fagpersoner om REST-integrasjoner, datamodeller og arbeidsflyter.
- Backend- og pipelinearbeid når teamet trenger det.

## Project

Tilde is Miljødirektoratet’s new platform for replacing older case-management systems within pollution and environmental administration. The solution is built as microservices on Azure, with services running in Docker containers on Kubernetes and delivery handled through GitHub Actions and FluxCD.

The platform covers several user surfaces: external users submitting applications and reports, case handlers processing cases and inspections, and administrators managing forms, code lists and access.

## Role and contribution

- Main focus on frontend development with Angular, TypeScript and SCSS.
- User interfaces for applications, reporting, inspections and case handling across the portals.
- Components and features aligned with Miljødirektoratet’s design system and shared Angular libraries.
- Collaboration with backend developers and domain experts on REST integrations, data models and workflows.
- Backend and pipeline work when the team needs it.
:::
