## Prosjekt

Holocaustsenteret driftet nettstedene hlsenteret.no og folkemord.no, som brukes til forskning, undervisningsmateriell og dokumentasjon om Holocaust og andre folkemord. Løsningen var bygget i WordPress med kartintegrasjon fra Geodata og var først driftet i AWS før den senere ble flyttet til et on-premise miljø.

## Rolle og bidrag

- Feilsøking og feilretting knyttet til hosting og driftsproblemer i AWS.
- PHP- og WordPress-kode, inkludert feilretting på serversiden i Apache.
- Oppdateringer og forbedringer i JavaScript- og jQuery-komponenter.
- Bidrag til stabil drift av digitale læringsressurser med høy tilgjengelighet.
:::

:::en
## Project

Holocaustsenteret operated hlsenteret.no and folkemord.no, sites used for research, teaching material and documentation about the Holocaust and other genocides. The solution was built in WordPress with map integration from Geodata and was first hosted on AWS before later moving on-premise.

## Role and contribution

- Troubleshooting and fixes for hosting and operational issues on AWS.
- PHP and WordPress code, including server-side fixes in Apache.
- Updates and improvements to JavaScript and jQuery components.
- Contribution to stable operation of digital learning resources that needed high availability.
